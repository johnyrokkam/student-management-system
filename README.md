# student-management-system
Java console-based CRUD application for student record management
## Features
- Add student record
- View all students
- Search student by ID
- Update student details
- Delete student record
- Handles empty record scenarios

## Technologies Used
- Java
- OOP (Object-Oriented Programming)
- Java Collections (ArrayList)
- VS Code

## Concepts Applied
- CRUD Operations
- Data handling
- Modular programming
- Exception-aware logic

## How to Run
1. Compile:
javac Student.java Main.java

2. Run:
java Main
